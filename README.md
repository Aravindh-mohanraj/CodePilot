# CodePilot
Automated Programming Question generation using AI

## 🚀 Overview

CodePilot is an AI-powered coding interview preparation platform that uses the **Gemini API** to generate coding problems based on the selected **company**, **difficulty**, and **topic**. It provides detailed problem statements, optimized **Python** and **Java** solutions, sample test cases, and supports downloading the generated content in **JSON** format for reuse across other coding platforms.

## ✨ Features

* 📌 Filter problems by **Company**, **Difficulty**, and **Topic**
* 🤖 AI-generated coding problems using the **Gemini API**
* 🗂️ Custom local dataset for relevant interview questions
* 💻 Optimized solutions in **Python** and **Java**
* ✅ Sample test cases with expected outputs
* 📥 Download problems and solutions as **JSON**
* ⚡ Fast and user-friendly interface

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Backend:** Python (Flask/FastAPI)
* **AI:** Gemini API
* **Database:** 
* **Export Format:** JSON

## 📖 How It Works

1. Select a **Company**.
2. Choose a **Difficulty** level.
3. Select a **Topic**.
4. Generate a coding problem.
5. View the problem statement, Python and Java solutions, and sample test cases.
6. Download the generated content as a JSON file and use in other Platforms.
